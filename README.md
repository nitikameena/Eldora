# Eldora
Eldora is a web-based OS where users can open apps like Notes, Files, Gallery, and To-Do in separate windows. It includes features like a live to-do list, image uploads, app search, wallpaper customization, and a taskbar with time and open apps. Overall, it’s a simple, interactive desktop operating system.
